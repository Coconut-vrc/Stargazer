# docs

設計書・開発向けドキュメント・モックをまとめたフォルダです。

| ファイル／フォルダ | 内容 |
|--------------------|------|
| **アプリ仕様書.md** | 見積・受託用の正式なアプリケーション仕様書（画面・データ・非機能・API制限・実装方式・制約） |
| **FUNCTIONAL_SPEC.md** | 機能設計書（画面一覧・操作手順・データフロー等） |
| **SECURITY_GUIDE.md** | セキュリティ対策の説明（店長向け・運用者向け） |
| **mock/** | 開発・検証用のサンプル（HTML・スクリーンショット等） |

ルートの `README.md` にはプロジェクト概要とデプロイブランチの記載があります。
