### よく見られる質問項目
- キャスト指名（第1希望・第2希望）
- RPの希望内容・シチュエーション
- 参加人数（1名 or 2名）
- 意気込み・質問欄
- 苦手な行為の申告
- パートナーへの承諾確認

### 共通の注意事項・禁止事項
✅ **技術要件**
- アバター容量制限: 40～50MB以下
- テクスチャメモリ: 150MB以下
- 身長制限: 165cm以下が多い
- 人型アバター推奨

❌ **禁止事項**
- 無許可での配信・録画・録音
- NSFW該当アバターの使用
- 版権アバターの使用
- 過度なパーティクル使用
- キャスト/スタッフへのフレンド申請
- 荒らし・誹謗中傷行為
- 無言参加（日本語VC必須）

⚠️ **その他共通ルール**
- RP（ロールプレイ）イベントが多い
- キャストはVRC名でなく名札の名前で呼ぶ
- 素性を聞き出す行為の禁止
- 終了時刻は前後する可能性あり
