# レイアウトモック（サンプル）

このフォルダは **現状のアプリレイアウトを静的な HTML で再現したモック** です。

## 目的

- レイアウト崩れやスタイルのずれを確認するときの **参照用**
- 本番の React アプリと同じ CSS（`common.css`・`layout.css`）を読み込んでいるため、見た目を揃えやすい
- 崩れ等があれば、このモックを基準に修正方針を決める

## ファイル

| ファイル | 内容 |
|----------|------|
| `layout.html` | アプリ全体の骨組み（サイドバー + メインエリア・テーマ切替・ログアウト） |
| `lottery-condition.html` | 抽選条件ページのモック（営業モード・当選者数・総テーブル数・マッチング方式） |
| `matching-result.html` | マッチング結果ページのモック（当選者別テーブル・キャスト別テーブル・エクスポートボタン） |

## 使い方

1. 各 HTML をブラウザで開く（`file://` またはローカルサーバー）
2. CSS は `../../common.css` と `../../css/layout.css` を参照しているため、**`app` フォルダをルートにした状態**で開く必要がある  
   - 例: `npx serve app` で `http://localhost:3000/mock/sample/layout.html` のようにアクセス
3. レイアウト修正時は、このモックと実装のクラス名・構造を揃えると保守しやすい

## 注意

- モックは静的な HTML のため、状態やルーティングは再現していません
- 本番と同じクラス名・セマンティクスをできるだけ使っています
